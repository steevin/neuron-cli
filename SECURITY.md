# Security Policy

If you find a security bug or vulnerability in Neuron CLI, please don't open a public issue. Instead, report it privately so we can patch it before it's public.

## How to report a security issue

Please send an email directly to:
* **[neuron@steevin.com](mailto:neuron@steevin.com)**

Try to include:
1. What the vulnerability is and what it affects.
2. Steps to reproduce it (or a quick proof of concept).

## What happens next

We'll look into it, acknowledge your email within 48 hours, and work on a fix as fast as we can. Once the fix is out, we can disclose it publicly and credit you for finding it (unless you prefer to remain anonymous).

Thanks for helping keep Neuron CLI secure!
